Propiedades en el contenedor Grid

Lo primero que tenemos que hacer es definir su estructura

Para definir su estructura utilizaré las siguiente propiedades (tras haber usado la propiedad display:grid)

  • grid-template-columns: Número de columnas del grid y tamaño de las mismas
  • grid-template-rows: Número de filas del grid y tamaño de las mismas
  • grid-row-gap: Separación entre las filas
  • grid-column-gap: Separación entre las columnas

En estas propiedades puedo tener varias formas de expresarlas, vamos a verlas:

COLUMNAS
  • grid-template-columns: Debemos de poner tantos valores de anchura como columnas quiero que tenga el GRID
  • grid-template-columns: 20% 50% 30% (Estoy diciendo que tengo 3 columnas y cada % es el tamaño de la 1 a la 3)
  • grid-template-columns: 100px auto 100px 100px
  • grid-template-columns: auto auto auto auto auto
  • grid-template-columns: [id] 100px [nombre] 300px [apellidos] auto (Estamos hablando aquí de que tenemos 3 columnas a las que les hemos dado nombre y tamaño)
FILAS
  • grid-template-rows: Debemos de poner tantos valores de altura como filas quiero que tenga el GRID
  • grid-template-rows: 20% 50% 30% (Estoy diciendo que tengo 3 filas y cada % es el tamaño de la 1 a la 3)
  • grid-template-rows: 100px auto 100px 100px
  • grid-template-rows: auto auto auto auto auto
  • grid-template-rows: [uno] 100px [dos] 300px [tres] auto (Estamos hablando aquí de que tenemos 3 filas a las que les hemos dado nombre y tamaño)
REPETIR VALORES Y UNIDAD FR

grid-template-column: repeat(3, 20% [col-start]) auto -> Indicamos 3 columnas con 20% de ancho y nombre "col-start" y luego una cuarta columna con ancho auto

grid-template-column: 2fr 100px 1fr 2fr (con la unidad fr, añadimos peso a la repartición del espacio sobrante)

Alineación horizontal DEL CONTENIDO

Utilizo la propiedad justify-items que posee los siguientes valores:

  • start
  • end
  • center
  • stretch (por defecto, el contenido ocupa toda la celda de mi grid)

Alineación vertical DEL CONTENIDO

Utilizo la propiedad align-items que posee los siguientes valores:

  • start
  • end
  • center
  • stretch (por defecto, el contenido ocupa toda la celda de mi grid)

Puedo juntar las dos alineaciones en una sola línea con place-item

place-item: start (vertical) end (horizontal)

Distribución horizontal dentro del contenedor cuando el grid no ocupa todo su contenedor

Esto lo hacemos con la propiedad justify-content

Distribución vertical dentro del contenedor cuando el grid no ocupa todo su contenedor

Esto lo hacemos con la propiedad align-content

Puedo juntar las dos distribuciones en una sola línea con place-content

place-content: start (vertical) end (horizontal)
