Elementos de la maquetación con CSS Grid

¿Que es Grid?

Maquetar con Grid es basicamente que vamos a tener un contenedor Grid gracias a la propiedad display:grid y con ello vamos a poder distribuir los elementos que están dentro de este contenedor.

Al contrario que los contenedores Flex, estas distribuciones no son solo en una dirección sino que son más complejas

Elementos

Como hemos dicho antes tendremos dos elementos en la maquetación con Grid: Contenedor y elementos grid

  • El contenedor GRID (display:grid o display: inline-grid)
  • Los elementos grid que están dentro y cuyo tamaño y posición vamos a modificar

¿Que propiedades voy a poder controlar desde el contenedor GRID?

  • Estructura en filas y columnas y la separación entre ellas
  • Definir áreas del GRID con nombre
  • La alineación horizontal y vertical de los elementos del GRID y del propio GRID

En definitiva, son propiedades que me van a permitir maquetar de una manera más sencilla a como se hace con el método tradicional
